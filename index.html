    Matching Game

    An online multiplayer matching game. I built the front end game using React & Phaser. I used Vite to build the React code. I developed the server using Node, Express, and Socket.io. I am also planning on integrating a MongoDB database to store user data. The game is deployed on Render and with the front end hosted on GitHub Pages. It was a challenge and lot of fun integrating Phaser with React, since there are not a lot of updated resources on the matter out there. I ended up using redux to manage the game states. Using redux allowed Phaser and React to communicated with each other. I used Socket.io to create the multiplayer functionality, to create game sessions, and to update the game for all players. The game is still in development, but I am planning on adding more features to it. I am also planning on adding a chat feature to the game.

    Technologies used:
    React, Phaser, Socket.io, NodeJS, Express
    Play The Game GitHub
